EM estimate vs direct optimization issue¶

Here I compare EM with optim in a larger simulated data-set involving 50 conditions. Example below is when EM and optim results are different and optim result seems better.

In [41]:
dat = readRDS('lite_data_1_artificial_mixture_1_mnm_shared_1.rds')

This data-set can be found in mmbr/inst/prototypes. Script to reproduce msusie analysis is available via cat(dat$DSC_DEBUG$script). Simply copy that code and run the data generation; then run the analysis using EM and optim. I save the X, Y, meta and resid_Y data into em_optim_difference.rds file for ease with loading it for further investigation.
